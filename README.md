# Stan Meyer Water Fuel System — AI Assisted Analysis and Documentation

This repository organizes Stanley Meyer water fuel system source material for analysis and eventual replication guidance. Primary technical focus areas include the Voltage Intensifier Circuit (VIC), water fuel cell, hydrogen airdation injection system, and related safety and bench-test procedures.

Referenced by: all project markdown files link back here via their **Referenced by** sections.

## Working Archive

```text
Stan_Meyer_Source_Archive/
  01_ocr_searchable_pdfs/canonical_file_names/   S001–S034 searchable PDFs
  02_extracted_text/                             S###_extract.txt
  03_figures_and_diagrams/                       F001–F004
  06_reference_documents/                        R001–R007
  00_originals/                                  historical only — not indexed (gitignored)
```
